## 📋 What this tool does

SkillDock serves as a bridge for your AI agents. It simplifies how you interact with complex server tools.

- **Centralized storage**: View all your installed skills in one window.
- **Git-aware updates**: The app notices when the original creators update a skill. It lets you apply these updates while keeping your custom changes intact.
- **Sync capability**: You can sync your setup across different folders or environments.
- **Compatibility**: The tool works with popular assistants like Claude, Cursor, and Windsurf, along with many MCP tools.

## ⚙️ Using the dashboard

When you open the app, you see a list of your current skills. Each entry shows the name, current status, and version.

To add a new skill, click the Add button in the top corner. Paste the link to the skill repository. The app downloads the files and sets them up for use.

To update an existing skill, click the checkmark icon next to the name. The app compares your version with the online version. If a update exists, the app prompts you to install it.

## ❓ Frequently asked questions

**Do I need to know how to code?**
No. You do not need any coding knowledge to use this tool. The interface handles the technical steps for you.

**Is my data private?**
Yes. SkillDock stores files locally on your machine. The app only communicates with the internet when you ask it to fetch updates or sync files.

**How do I delete a skill?**
Select the skill from your list. Click the trash icon. The app removes the files from your computer.

**The app does not see my AI agent, what should I do?**
Check your settings tab. Ensure the path to your AI tool is correct. Refresh your list by clicking the reload icon.

## 🛠️ Advanced management

You can create groups for your skills. Use the folder icon to organize your items by project or purpose. This makes it easier to find what you need when you manage many assistants.

The app keeps a history of your changes. If you make a mistake, you can revert to a previous version of a skill. Click the history icon to see your past edits. You can choose any previous point to restore your settings.

## 🌐 Connectivity and sync

The tool uses Git technology under the hood. This ensures your updates remain stable. When you sync, the app fetches new data only for the parts that changed. This saves time and bandwidth.

If you use multiple computers, copy your settings folder to a shared drive. Point SkillDock to that folder on each machine. Your skill list will stay consistent across all your workspaces.

## 🛑 Troubleshooting

If the app fails to start, restart your computer. Ensure your antivirus does not block the application. You can add an exception for the SkillDock folder in your security settings if the app remains blocked.

Check the logs folder to see if an error shows up. The logs give simple descriptions of any crashes or failed network connections. If you still have trouble, check the main repository page for known issues or new versions.

## 🚀 Future updates

We improve the tool based on user feedback. Updates occur once or twice per month. Open the app to see a notification when a new version arrives. Click the update button to install the latest features. Your current skill list remains untouched during these updates.
